# Chapter 1: Get yourself setup

Need a cluster to work with
- Try the [official docs](https://kubernetes.io/docs/tutorials/hello-minikube/) with minikube
- Need `kubectl`

```bash
# starts the cluster and setups the config for kubectl as the default namespace
minikube start

# Test out the cluster with this command
kubectl create deployment hello-node --image=registry.k8s.io/e2e-test-images/agnhost:2.53 -- /agnhost netexec --http-port=8080

# Verify that everything is working with these
kubectl get nodes # should see one node
kubectl get pods  # should see the one deployment that you just made
```

Could also try this setup with podman or something else. That stuff really isn't as important though
